# 🏦 Bank Simulator (C++)

A console-based banking system implemented using Object-Oriented Programming principles in C++.  
The project simulates core banking operations such as user authentication, account management, and financial transactions.

## Features

- User Registration & Login system  
- Password hashing with random salt generation  
- Savings and Current account support  
- Deposit and Withdrawal operations  
- Account creation and deletion  
- Polymorphic handling of multiple account types  

## Concepts Demonstrated

- Object Oriented Programming (Inheritance, Polymorphism, Abstraction)  
- Dynamic Memory Management  
- STL Containers (vector)  
- Basic Security (Salt + Hashing)  
- Modular System Design  

## ⚙️ How to Run

```bash
g++ main.cpp -o bank
./bank
